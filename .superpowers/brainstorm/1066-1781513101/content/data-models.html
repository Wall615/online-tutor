👤 User（用户表）
id (PK)
username
email
password_hash
role: student/teacher/parent/admin
phone
avatar
created_at
👨‍🏫 TeacherProfile（教师资料）
id (PK)
user_id (FK → User)
subjects
bio
hourly_rate
education

rating_avg
📚 Course（课程表）
id (PK)
teacher_id (FK → User)
title
subject
description
price
duration (分钟)
status: active/inactive
created_at
📅 Booking（预约表）
id (PK)
student_id (FK → User)
course_id (FK → Course)
scheduled_time
status: pending/confirmed/cancelled/completed
created_at
💬 Message（消息表）
id (PK)
sender_id (FK → User)
receiver_id (FK → User)
booking_id (FK → Booking)
content
sent_at
⭐ Review（评价表）
id (PK)
booking_id (FK → Booking)
student_id (FK → User)
teacher_id (FK → User)
rating: 1-5
comment
created_at
👨‍👩‍👧 ParentStudent（绑定表）
id (PK)
parent_id (FK → User)
student_id (FK → User)
bound_at
💰 Payment（支付表）
id (PK)
booking_id (FK → Booking)
student_id (FK → User)
amount
status: pending/paid/refunded
paid_at
🔗 关键关联：
User(teacher) 1→1 TeacherProfile  |  Teacher 1→N Course  |  Course 1→N Booking  |  Booking 1→1 Payment  |  Booking 1→1 Review  |  Booking 1→N Message  |  Parent N→N Student (via ParentStudent)
